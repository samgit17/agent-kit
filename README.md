# agent-kit 🤖

Pre-built LangGraph agent templates for indie hackers and micro-SaaS builders.

## What is this?
Stop building agent scaffolding from scratch. AgentKit gives you 
ready-to-run multi-agent workflows you can drop into your project.

## Why agent-kit?
LangGraph is powerful but the setup cost kills momentum. 
AgentKit removes that tax.

## Templates (coming soon)
- 🔍 **Research Agent** — deep-dive any topic, returns structured summary
- ✍️ **Content Draft Agent** — turns bullet points into publish-ready posts
- 🧩 **Task Breakdown Agent** — splits a goal into actionable subtasks

## Stack
- LangGraph · Python · Ollama · Docker
